# fcitx-vinput-rs

Demo Rust-oriented rewrite workspace for fcitx5-vinput.

This repository will start small and grow step by step:

- Rust protocol/config/daemon crates first.
- A thin C++ Fcitx5 addon bridge later, unless a usable Rust addon binding appears.
- Plans live under `docs/plan/` locally, but are intentionally ignored by git.
